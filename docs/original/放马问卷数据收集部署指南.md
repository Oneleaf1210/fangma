# 放马问卷数据收集部署指南

## 📋 概述

本指南将帮助您部署完整的问卷系统，包含数据收集功能。问卷数据将自动保存到Google Sheets，同时也会在用户浏览器本地备份。

## 🚀 快速开始

### 第1步：部署问卷到GitHub Pages

1. **创建GitHub仓库**
   ```bash
   # 1. 访问 https://github.com/new
   # 2. 仓库名: fangma-survey-data
   # 3. 选择 Public
   # 4. 点击 Create repository
   ```

2. **上传问卷文件**
   ```bash
   # 将以下文件上传到仓库：
   # - 放马产品市场调研问卷_带数据收集.html
   ```

3. **启用GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source: 选择 main 分支，/ (root) 目录
   - 点击 Save

4. **访问问卷**
   - 链接: `https://[您的用户名].github.io/fangma-survey-data/放马产品市场调研问卷_带数据收集.html`

### 第2步：设置Google Sheets数据收集（推荐）

#### A. 创建Google表格
1. 访问 [sheet.new](https://sheet.new) 创建新表格
2. 重命名表格为"放马问卷数据"
3. 在第一行添加表头（复制以下内容到A1-O1）：

```
提交时间 | 年龄 | 职业 | 压力程度 | 概念理解 | 学习兴趣 | 语言偏好 | 功能偏好 | 放松时长 | 使用场景 | 价格接受度 | 付费功能 | 推荐意愿 | 建议 | 联系方式
```

#### B. 创建Google Apps Script
1. 在Google表格中：**扩展程序 → Apps Script**
2. 删除默认代码，粘贴以下代码：

```javascript
function doPost(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const data = JSON.parse(e.postData.contents);

    // 整理数据为一行
    const row = [
      new Date(),
      data.age || '',
      data.occupation || '',
      data.stressLevel || '',
      data.conceptUnderstanding || '',
      data.learningInterest || '',
      data.languagePreference || '',
      data.featurePreference ? data.featurePreference.join(';') : '',
      data.relaxDuration || '',
      data.usageScenario ? data.usageScenario.join(';') : '',
      data.priceAcceptance || '',
      data.paidFeatures ? data.paidFeatures.join(';') : '',
      data.recommendation || '',
      data.suggestions || '',
      data.contact || ''
    ];

    sheet.appendRow(row);

    // 返回成功响应
    return ContentService
      .createTextOutput(JSON.stringify({ success: true, message: '数据保存成功' }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    // 返回错误响应
    return ContentService
      .createTextOutput(JSON.stringify({
        success: false,
        message: '保存失败: ' + error.message
      }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// 测试函数
```

3. **保存项目**
   - 点击保存按钮（Ctrl+S）
   - 项目名称: `放马问卷数据收集`

#### C. 部署为Web应用
1. 点击 **部署 → 新建部署**
2. 类型选择：**Web 应用**
3. 配置设置：
   - 说明：放马问卷数据收集API
   - 执行身份：**我** (您的Google账号)
   - 访问权限：**任何人** (重要！)
4. 点击 **部署**
5. 授权Google账号（点击"Review Permissions" → 选择您的账号 → "Allow"）
6. **复制Web应用URL** (格式: `https://script.google.com/macros/s/XXXXXXXXXX/exec`)

#### D. 配置问卷
1. 打开 `放马产品市场调研问卷_带数据收集.html`
2. 找到第875行附近的配置区域：
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```
3. 将 `YOUR_SCRIPT_ID` 替换为您的实际Script ID
4. 保存文件并重新上传到GitHub

## 🔧 故障排除

### 常见问题

#### 1. Google Script返回403错误
**原因**：未正确设置访问权限
**解决**：
- 重新部署Web应用
- 确保"访问权限"设置为"任何人"
- 重新授权Google账号

#### 2. 数据未保存到Google Sheets
**检查步骤**：
1. 打开浏览器开发者工具（F12）
2. 查看Console标签中的错误信息
3. 检查Network标签中的请求状态
4. 确保Google Script URL正确

#### 3. 表单提交卡住
**可能原因**：
- 网络问题
- Google Script执行超时
- 浏览器插件干扰

**解决**：
- 刷新页面重试
- 检查网络连接
- 禁用广告拦截插件

#### 4. 本地存储已满
**解决**：
1. 点击"导出本地数据"按钮备份
2. 清除浏览器本地存储：
   ```javascript
   localStorage.removeItem('fangma_survey_responses');
   ```

## 📈 数据分析建议

### 基础分析（使用Google Sheets函数）
```excel
# 1. 统计年龄分布
=COUNTIF(B:B, "18-24")  # B列是年龄

# 2. 计算平均推荐意愿
=AVERAGE(M:M)  # M列是推荐意愿

# 3. 统计最受欢迎的功能
# 先拆分功能偏好列（分号分隔）
=SPLIT(H2, ";")  # H列是功能偏好

# 4. 创建数据透视表
插入 → 数据透视表
```

### 高级分析（导出到Excel/Google Data Studio）
1. **用户画像分析**：年龄、职业、压力程度分布
2. **功能偏好分析**：各功能选择频率
3. **定价分析**：价格接受度分布
4. **相关性分析**：职业 vs 功能偏好，年龄 vs 语言偏好等

## 🔒 隐私和安全

### 数据保护措施
1. **最小化收集**：只收集必要信息
2. **匿名处理**：不收集姓名、身份证等敏感信息
3. **加密传输**：HTTPS协议保护数据传输
4. **访问控制**：Google表格可设置查看/编辑权限

### 合规性说明
在问卷开头添加隐私声明：
```
隐私声明：
1. 本问卷数据仅用于"放马"产品研究和优化
2. 收集的信息包括：年龄、职业、偏好等基本信息
3. 数据将匿名处理，不会关联到个人身份
4. 您可以选择不留下联系方式
5. 调研结束后，数据将妥善处理或匿名化保存
```
